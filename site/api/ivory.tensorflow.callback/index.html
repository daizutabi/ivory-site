ivory.tensorflow.callback

ivory.tensorflow.callback.Callback(run)

Callback class used to connect Ivory and Tensorflow's original Callback.

Parameters

METHOD
on_train_batch_begin(batch, logs=None)

Called at the beginning of a training batch in fit methods.

Subclasses should override for any actions to run.

Parameters
  • batch — integer, index of batch within the current epoch.
  • logs (, optional) — dict. Has keys batch and size representing the current batch number and the size of the batch.
METHOD
on_test_batch_begin(batch, logs=None)

Called at the beginning of a batch in evaluate methods.

Also called at the beginning of a validation batch in the fit methods, if validation data is provided.

Subclasses should override for any actions to run.

Parameters
  • batch — integer, index of batch within the current epoch.
  • logs (, optional) — dict. Has keys batch and size representing the current batch number and the size of the batch.
